# Product Brief

## Vision

Thaumacord helps organizers run social live-action games where rules, secrecy, resources, roles, votes, and timed events are hard to manage on paper.

The app should feel like an invisible game master assistant and a set of digital props rather than a normal mobile form interface.

## Core Users

- Game master: prepares, launches, arbitrates, corrects, and reviews the game.
- Player: receives private role information, resources, cards, actions, messages, and prompts.
- Module creator: converts a rules document into a reusable game module.
- Admin/developer: maintains modules, releases builds, and inspects logs.

## Design Principles

- Phone as prop, not only screen.
- Secret by default: every view is filtered by visibility rights.
- The game master can always override.
- Every important action is logged.
- Rules are configurable before they are clever.
- Fragile sensor actions must always have a manual fallback.

## Non-MVP Roadmap Themes

- Physical gestures using NFC, BLE, sensors, vibration, QR fallback.
- Real-world zones mapped to imaginary places.
- Mandragore AI API integration for rules, strategy, arbitration, and module creation.
